# Diagonal Difference Algorithm

## Description
Given a square matrix of size n×n, this project calculates the absolute 
difference between the sums of its two diagonals.

## Algorithms Implemented
- **Iterative (Non-Recursive):** Uses a single loop — O(n) time, O(1) space
- **Recursive:** Uses a recursive helper function — O(n) time, O(n) space

## Language
C++

## Files
- `iterative.cpp` — Non-recursive implementation
- `recursive.cpp` — Recursive implementation
- `Diagonal_Difference.docx` — Full documentation including pseudo-code, 
   analysis, complexity, and comparison
